# OpenProject Telegram Bot

A Telegram bot for integrating with OpenProject that provides instant notifications in chat.

## Core Functionality

The bot receives webhooks (notifications) from OpenProject about any updates (new tasks, status changes, comments, etc.) and automatically sends personalized notifications to Telegram.

## Who Receives Notifications?

Notifications are sent only to users who are associated with the project or specific task where the change occurred.

## Why Use This Bot?

*   **Instant Alerts:** Get notifications about important events directly in Telegram.
*   **Personalized Distribution:** Each user sees only relevant notifications for them.
*   **Communication Centralization:** Reduces email clutter and helps keep the team informed about changes.
